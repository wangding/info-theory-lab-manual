# 实验一、自信息量函数曲线绘制

## 实验内容

分别用 Excel 和 Matlab 软件绘制自信息函数曲线。

## 实验环境

1. 计算机
2. Windows 2000 或以上
3. Microsoft Office XP 或以上
4. Matlab 6.5 或以上

## 实验目的

1. 掌握 Excel 的数据填充、公式运算和图表制作
2. 掌握 Matlab 绘图函数
3. 掌握、理解自信息函数表达式及其性质
